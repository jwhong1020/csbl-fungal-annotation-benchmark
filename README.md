# Project Objectives

This project aims to establish a reliable genome annotation for a bioluminescent fungus and use the selected annotation for downstream transcriptome analysis.

The project consists of two connected stages.
## Background

## Stage 1. Genome annotation evaluation

BRAKER4, FunGAP, and Helixer will be applied independently to:

- the target bioluminescent fungus
- Saccharomyces cerevisiae S288C

For each organism, the structural annotation results generated by the three programs will be compared.

S. cerevisiae S288C is included as a model organism with an established reference annotation. It is used to evaluate the output characteristics and prediction accuracy of each annotation program, not for comparative genomics with the bioluminescent fungus.

For the target fungus, the annotation results will be evaluated using:

1. gene and transcript counts
2. exon, intron, CDS, and protein length distributions
3. BUSCO completeness
4. RNA-seq evidence support
5. protein homology support
6. agreement among annotation programs
7. recovery of known fungal bioluminescence genes

Based on these results, a final gene annotation will be selected for downstream analysis.

## Stage 2. Differential expression and co-expression analysis

The selected genome annotation will be used as the reference gene set for RNA-seq expression analysis.

The downstream analysis will include:

1. RNA-seq quality control
2. Read alignment or transcript quantification
3. Gene-level count matrix construction
4. Differentially expressed gene analysis between experimental conditions
5. Identification of genes showing positive or negative expression correlation with known bioluminescence genes
6. Functional annotation and biological interpretation of candidate genes

Known fungal bioluminescence genes, such as luz, hispS, h3h, and cph, will be used as reference genes for expression-pattern analysis.

Differential expression analysis and co-expression analysis address different questions:

- DEG analysis identifies genes whose expression levels differ significantly between conditions.
- Co-expression analysis identifies genes whose expression patterns resemble or oppose those of bioluminescence-related genes across samples.

Genes supported by both analyses will be prioritized as candidate genes associated with variation in fungal bioluminescence.
