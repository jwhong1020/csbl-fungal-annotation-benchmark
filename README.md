# Project Objectives

This project aims to establish a reliable genome annotation for a bioluminescent fungus and use the selected annotation for downstream transcriptome analysis.

The project consists of two connected stages.  
1. Evaluation and comparison of structural genome annotations generated by BRAKER4, FunGAP, and Helixer
2. Identification of genes co-expressed with known fungal bioluminescence genes

Evaluation and comparison of structural genome annotations generated by BRAKER4, FunGAP, and Helixer
Identification of genes co-expressed with known fungal bioluminescence genes

BRAKER4, FunGAP, and Helixer will be applied independently to the target bioluminescent fungus and to Saccharomyces cerevisiae S288C.

S. cerevisiae S288C is included as a model organism with a well-established reference annotation. It is used to evaluate the behavior and accuracy of each genome annotation pipeline. The project does not compare the genomes of S288C and the bioluminescent fungus.  

# Objectives
## Objective 1. Structural genome annotation comparison

Structural genome annotations will be generated using:  

- BRAKER4
- FunGAP
- Helixer

The outputs will be compared within each organism based on:

Number of predicted genes and transcripts
Gene, CDS, exon, intron, and protein length distributions
Number of exons per gene
Proportion of single-exon genes
BUSCO completeness
RNA-seq evidence support
Protein homology support
Agreement among annotation programs

For S. cerevisiae S288C, the predicted annotations will additionally be compared with the established reference annotation.

For the target bioluminescent fungus, the results will be used to select a final gene annotation suitable for downstream RNA-seq expression analysis.

## Objective 2. Identification of bioluminescence-associated co-expression patterns

The selected gene annotation will be used to quantify gene expression across multiple RNA-seq samples.

Known fungal bioluminescence genes, including luz, hispS, h3h, and cph, will be used as reference genes for expression-pattern analysis.

The analysis will identify genes that:

1. Show expression profiles similar to known bioluminescence genes
2. Show expression profiles inversely correlated with known bioluminescence genes
3. Belong to expression modules containing bioluminescence genes
4. Encode proteins potentially related to transcriptional regulation, signaling, metabolism, transport, or stress response

Co-expression indicates statistical similarity between expression profiles. It does not by itself demonstrate direct regulation, physical interaction, or causal involvement in fungal bioluminescence.

# Analysis workflow  
```text
Genome assembly  
    ├── BRAKER4
    ├── FunGAP
    └── Helixer
            │
            ▼
Structural annotation comparison
            │
            ▼
Selection of the final annotation
            │
            ▼
RNA-seq expression quantification
            │
            ▼
Bioluminescence-gene-centered
co-expression analysis
            │
            ▼
Candidate gene prioritization
Co-expression analysis
```
Gene expression will be quantified across multiple biological samples, conditions, or time points.

Possible co-expression approaches include:

- Pearson correlation
- Spearman correlation
- Hierarchical clustering
- Time-series expression clustering
- Weighted gene co-expression network analysis

The final method will be selected according to the number of available samples and the experimental design.

A sufficiently large number of independent RNA-seq samples is required for reliable co-expression analysis. The currently available single RNA-seq sample cannot be used to calculate gene-to-gene expression correlations.
